Professional AI extension for Elementor that integrates ChatGPT capabilities into your WordPress website.

== Description ==

PK Elementor AI brings the power of artificial intelligence to your Elementor page builder. Generate high-quality content, get creative suggestions, and optimize your workflow using ChatGPT technology.

= Key Features =

* AI-powered content generation
* Smart content suggestions
* Seamless Elementor integration
* Multi-language support (EN, NL, DE, FR, ES)
* User-friendly interface
* Real-time preview
* Customizable AI parameters
* Secure API handling

= Pro Features =

* Advanced AI models
* Batch processing
* Custom training
* Priority support
* Extended API limits

= Languages =

PK Elementor AI is available in:

* English (Default)
* Dutch (Nederlands)
* German (Deutsch)
* French (Français)
* Spanish (Español)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pk-elementor-ai`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your API settings in the plugin dashboard
4. Start using the AI widgets in Elementor

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need an OpenAI API key to use the content generation features.

= Is it compatible with my theme? =

Yes, PK Elementor AI works with any WordPress theme that supports Elementor.
